Actividad Formulario HTML

Descripción:

Este proyecto consiste en el desarrollo de un formulario de inscripción para un Taller de Desarrollo Web utilizando únicamente HTML, sin emplear CSS ni JavaScript.

El formulario permite recopilar información personal, académica, preferencias del taller, disponibilidad e información adicional de los participantes.

Elementos HTML utilizados:

•	form

•	label

•	input

•	select

•	option

•	textarea

•	button

•	fieldset

•	legend

Tipos de input utilizados:
•	text

•	email

•	tel

•	number

•	date

•	time

•	url

•	radio

•	checkbox

•	file

•	color

•	range

Validaciones aplicadas:

•	required

•	placeholder

•	minlength

•	maxlength

•	min

•	max

•	pattern

•	step

Secciones del formulario:

1. Datos Personales

•	Nombres y apellido

•	Correo electrónico

•	Número de teléfono

•	Edad

•	Fecha de nacimiento

2. Datos Académicos
   
•	Carrera o curso

•	Nivel académico

•	Institución educativa

3. Preferencias del Taller

•	Modalidad del taller (Presencial, Virtual o Híbrida)

•	Temas de interés (HTML, CSS y JavaScript)

•	Nivel de conocimiento

•	Color favorito

•	Nivel de motivación

4. Disponibilidad

•	Fecha preferida para iniciar

•	Hora disponible

•	Jornada preferida (Mañana, Tarde o Noche)

5. Información Adicional

•	Subida de archivo

•	Portafolio o perfil académico

•	Comentarios adicionales
