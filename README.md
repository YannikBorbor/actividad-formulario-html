# Formulario de Inscripción al Taller de Desarrollo Web

## Descripción

Este proyecto consiste en el desarrollo de un formulario de inscripción para un Taller de Desarrollo Web utilizando únicamente HTML5. El formulario permite recopilar información personal, académica y preferencias de los estudiantes interesados en participar en el taller.

Además, incluye diferentes tipos de campos de entrada, listas desplegables, botones, áreas de texto y validaciones básicas de HTML para garantizar una correcta captura de datos.
